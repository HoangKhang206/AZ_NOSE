Bạn đang hỗ trợ dự án "AZ FACE INSIGHT" — một web-app phân tích tỷ lệ khuôn mặt kết hợp AI tư vấn, được xây dựng cho phòng khám AZ NOSE (chuyên sâu nâng mũi, aznose.vn). Dự án được làm trong khuôn khổ môn Tư duy Tính toán (CT) tại HCMUS, nhóm N07.

TÓM TẮT SẢN PHẨM:
- Phân hệ nhúng trong website aznose.vn (không phải web tách rời)
- User upload 3 ảnh (chính diện + nghiêng 45° + nghiêng 90°) → AI phân tích tỷ lệ khuôn mặt → gợi ý dáng mũi phù hợp → chatbot tư vấn → CTA đặt lịch chụp CT 3D miễn phí
- Traffic mục tiêu: khách từ TikTok/Facebook, 18-30 tuổi, 85-90% mobile

STACK KỸ THUẬT:
- Frontend: Next.js hoặc React SPA, hosting Vercel/Cloudflare Pages (free tier)
- Computer Vision: MediaPipe Face Mesh (Google, client-side, load qua CDN) → 468 landmark 3D
- Toán học: Định lý hàm Cosin + khoảng cách Euclidean → góc mũi môi, góc mũi trán, tính đối xứng
- AR: Silhouette overlay Cấp độ 2 (vẽ đường viền dáng mũi đề xuất chồng lên ảnh gốc, KHÔNG biến đổi ảnh)
- AI: Dual-RAG (2 tầng), dùng Gemini 1.5 Flash hoặc Groq (free tier)
  * AI-1 "Phân tích": sinh 3-4 câu tư vấn dựa trên kho tri thức bác sĩ
  * AI-2 "Ngọc": chatbot tư vấn kinh doanh dựa trên kho tri thức FAQ + danh mục dịch vụ
- Backend: Serverless Functions (chỉ proxy LLM API, không xử lý ảnh)
- Storage: Google Sheets API (MVP), CRM sau

4 SUB-PROBLEM (theo computational nature, KHÔNG theo UI):
1. Face Landmarking — Pre-trained CNN inference (MediaPipe)
2. Geometric Analysis — Euclidean + Cosine law
3. Silhouette AR Overlay — Canvas 2D + SVG anchored on landmark
4. Dual-RAG Conversational AI — LLM + Retrieval + Guardrails

- KHÔNG chẩn đoán y khoa cá nhân (Luật Khám chữa bệnh 2023 — chỉ bác sĩ được chẩn đoán)
- AI chỉ sinh nội dung dựa trên kho tri thức đã được duyệt (RAG cứng)
- Bắt buộc có guardrails regex + fallback template trước khi hiển thị output AI
